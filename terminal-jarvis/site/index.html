Getting Started

Set up in 3 steps

01

Install Ollama

Ollama runs LLMs locally. Install it first — it takes under a minute.

terminal 
# Linux / macOS
curl -fsSL https://ollama.com/install.sh | sh

# Windows — download from
# https://ollama.com/download/windows

After installing, verify with ollama --version and keep Ollama running in the background.

02

Pull a model

Jarvis works with any Ollama model. For agentic tool-calling, use qwen2.5. For general chat, any model works.

terminal 
# Recommended — supports tool calling (agentic)
ollama pull qwen2.5

# Alternatives
ollama pull llama3
ollama pull mistral
ollama pull gemma2
qwen2.5 — agentic llama3 — general mistral — fast gemma2 — compact
03

Download & launch Jarvis

Grab the build for your OS, extract, and run the included launcher. No npm install is required.

terminal 
# Linux
tar -xzf terminal-jarvis-linux-x64.tar.gz
cd terminal-jarvis-linux-x64
./run-jarvis.sh

# Windows — extract zip, then run:
run-jarvis.bat

First run: click connect vault to link Obsidian, then chat. Use save reply to append answers into Jarvis/YYYY-MM-DD.md inside your vault.
